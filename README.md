# 🏨 Hostel Management System

A Full Stack **Hostel Management System** built using the **MERN Stack** that enables administrators to efficiently manage hostels, rooms, beds, and future student allocations through a modern dashboard.

---

## 🚀 Features

### 🔐 Authentication
- Admin Login
- JWT Authentication
- Protected Routes
- Password Hashing using bcrypt

### 🏠 Hostel Management
- Create Hostel
- View All Hostels
- Update Hostel Details
- Delete Hostel

### 🚪 Room Management
- Create Rooms
- View All Rooms
- Update Room Information
- Delete Rooms

### 🛏 Bed Management
- Create Beds
- View Available Beds
- Delete Beds
- Bed Status (Available / Occupied)

### 📊 Dashboard
- Total Hostels
- Total Rooms
- Total Beds
- Clean Admin Dashboard
- Responsive UI

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/RishabhChouhan18/Hostel-Management-System.git
```

---

### Backend

```bash
cd Backend
npm install
npm run dev
```

---

### Frontend

```bash
cd Frontend/hostel-management
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the Backend folder.

```env
PORT=5000

MONGO_URI=YOUR_MONGODB_CONNECTION_STRING

JWT_SECRET=YOUR_SECRET_KEY
```
